Some more information:

What is the Chocolatey Community Repository?

This is a special section of the Chocolatey website that contains packages maintained and validated by our community that you can use for free to install and manage software. It's a great resource for the community, however if you are an organization there are some caveats to consider before using the community repository directly (such as rate limiting, reliability, etc).

You can also develop your own package and share it via the Chocolatey Community Repository.

What is Chocolatey?

Chocolatey is like apt-get, but for Windows.

How Does Chocolatey Work?

When you use Chocolatey to install a package, it copies the package to your system, runs the PowerShell script chocolateyInstall.ps1 (if there is one in the package) and puts executables on the path (if there are any in the package).
